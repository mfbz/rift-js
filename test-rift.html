Rift Protocol Test Page

This page contains a Rift link which should be detected by the Harpoon wallet extension.

How it works:

  1. Make sure your Rift widget is running on localhost:3000
  2. Make sure you have the Harpoon wallet extension installed
  3. Click the link below or wait for Harpoon to detect it automatically
  4. Approve the injection when prompted
  5. The widget should appear on this page inside an iframe
rift://localhost:3000

Troubleshooting:

  • If nothing happens, check if Harpoon wallet is properly installed
  • Ensure your widget is running at the correct URL
  • Check browser console for any errors
